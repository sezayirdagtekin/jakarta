### Run Locally

Clone the project

```bash
  git clone https://github.com/sezayirdagtekin/jakarta.git
```

Go to the project directory

```bash
  cd  bookstore
```

Install dependencies

```bash
  mvn clean install
```

Start the server

```bash
  ./mvnw  quarkus:dev
```

### Dev UI > Open Api
http://localhost:8080/q/dev-ui/
